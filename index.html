Gleam Box

Gleam Box

Description: A one-tap rhythmic runner game merging Flappy Bird, Snake, and Stardew Valley's fishing minigame.

  • Game Jam: UDC Game Jam #34
  • Theme: Three Modes = Three Colors
  • Features: Multi-input support, Discord Rich Presence, leaderboard, FPS monitor, upgrade shop
  • External Assets: Duet theme (Tim Shiell), Discord SDK, Leaderboard Creator
  • Collaborators: Muhammad Ali (90%), Finn Nijpels (8%), Blackbelt_mfer (2%)
  • Released: Jun 14, 2025
  • Platforms: Itch.io, Google Play, Unity Web
